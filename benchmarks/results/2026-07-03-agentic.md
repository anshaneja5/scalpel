# Agentic benchmark — 2026-07-03

Ponytail's agentic harness (vendored, patched only for portability and symmetric arm
activation): every cell is a real headless Claude Code session (v2.1.200) editing a
seeded workspace, scored on the files it leaves behind. Model: Haiku 4.5.
Arms activated identically — each skill's full SKILL.md via `--append-system-prompt`;
no plugin machinery for any arm. `Bash` disallowed, MCP stripped, `bypassPermissions`,
fresh repo copy per cell.

## LOC tier — 12 real features on full-stack-fastapi-template @ cd83fc1 (n=3/task, 36 cells/arm)

| arm | med src LOC | total cost | med time | correct | completeness (judge) | over-eng (judge) |
|---|--:|--:|--:|--:|--:|--:|
| baseline | 138.5 | $3.68 | 48.8s | 100% | 2.94 | 1.67 (22 cells ≥2) |
| ponytail | 43.5 | $2.68 | 37.6s | 100% | 3.00 | 1.19 (15 cells ≥2) |
| **scalpel** | **42.5** | $2.75 | **36.8s** | 100% | **3.00** | **1.17 (12 cells ≥2)** |

Judges are upstream's auditable LLM judges (Sonnet 4.6, temperature 0, published rubric,
selftest-validated). Completeness 3.00 with min 3 on every scalpel cell: the low LOC is
not stub-shipping.

**Honest verdict: a statistical tie with ponytail.** Scalpel edges LOC (−2%), time (−2%),
and over-engineering flags (12 vs 15); ponytail edges cost (+2.6% for scalpel). At n=3
none of these gaps clear the noise. Both arms cut ~69% of baseline's code. The single-shot
suite (same tasks ponytail publishes) is where scalpel separates — see
[2026-07-03-single-shot.md](2026-07-03-single-shot.md).

## Iteration notes (found by this benchmark, fixed in SKILL.md, re-measured)

- Scalpel initially built shadcn-style popover/calendar components — and touched
  `package.json` — on the native-input trap tasks (datepicker median 98 LOC). After the
  manifest red line + ~30-line UI-control tripwire: datepicker 24, colorpicker 24
  (ponytail: 64, 40). The star-rating task resists the fix for both arms (component-library
  idiom gravity); scalpel still runs heavy there (~95–112 vs ponytail ~70).
- Judge fix (upstream bug on LOC-tier fixtures): `source_text()` concatenated the whole
  repo clone, silently nulling every score; patched to judge the agent's `git diff`.

## Reproduce

```bash
cd benchmarks/agentic
python3 run.py --selftest     # no API
FSTMPL_DIR=/path/to/full-stack-fastapi-template python3 run.py \
  --task tmpl-fe-datepicker,...,tmpl-be-csv --arms baseline,ponytail,scalpel \
  --models haiku --runs 3 --workers 6
python3 complete.py --run runs/<stamp> && python3 judge.py --run runs/<stamp>
```
